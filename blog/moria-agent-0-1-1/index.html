Moria Agent 0.1.1 — First Release

Moria Agent debuts with Spotlight access, connected context, structured output, persistent sessions, and controlled execution.

By Moria Agent Team

Moria Agent 0.1.1 is the first product release of an AI workspace designed to stay close to your work, connect with the context around it, and turn complex requests into results that are ready to use.

Everywhere. Beautifully structured.

AI where you need it

Moria Launcher brings the agent into your current flow. Open Spotlight from another application, capture relevant context, and start or continue a session without switching away from the task in front of you.

The first packaged desktop release is available for Windows.

Results designed for action

Moria Agent goes beyond plain generated text. Responses can become:

  • Clearly organized explanations and implementation plans
  • Structured documents, tables, and spreadsheets
  • Diagrams and formatted code changes
  • Rich previews for HTML, PDFs, images, and Markdown

The presentation adapts to the work, making results easier to inspect, review, and share.

Context from across your workflow

Connect local and online information through MCP servers, REST APIs, local files, and browser workflows. A session can bring together repositories, issue trackers, documents, conversations, internal services, and files on your computer—then organize that context into a coherent result.

Persistent, organized work

Sessions preserve their history, decisions, plans, and output. They are organized like an inbox and task manager, so you can move between tasks without losing context.

Multiple sessions can run independently, allowing longer work to continue in the background while you focus elsewhere.

Control built in

Choose the level of autonomy for each task:

  • Explore researches and analyzes without modifying your work.
  • Ask to Edit requests approval before individual changes.
  • Execute completes an approved task autonomously.

Plans remain reviewable before implementation, keeping people in control of consequential work.

Workspaces that fit the job

Workspaces isolate sources, skills, sessions, permissions, and configuration for different projects or responsibilities. Statuses and labels shape the workflow, while themes and reusable skills adapt Moria Agent to the way each team works.

Available now

Moria Agent 0.1.1 establishes the foundation for connected, context-aware agent workflows on the desktop, in the browser, and from the command line.
