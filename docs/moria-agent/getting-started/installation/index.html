Installation

Install the Moria Agent desktop app on macOS, Windows, or Linux

Moria Agent is a native desktop application. It runs locally on your machine, stores credentials encrypted on disk, and connects to external services only when you configure sources.

System Requirements

PlatformArchitecturesNotes
macOSApple Silicon (arm64)macOS 11+. Intel Macs are not supported from v0.10.1 onward (v0.10.0 was the final Intel build).
Windowsx64Windows 10+
Linuxx64AppImage format, requires FUSE

Note

macOS Intel (x64) builds are discontinued as of v0.10.1. New releases ship for Apple Silicon (M-series) only. Intel Macs can still install the final supported build, v0.10.0, but will not receive future updates. Windows and Linux are unaffected.

Manual Install (Recommended)

Apple Silicon (M1/M2/M3/M4):

  1. Download the .dmg
  2. Open the DMG and drag Moria Agent to Applications
  3. Launch from Applications or Spotlight (CMD+Space, type "Moria Agent")

Note

Intel Macs are not supported. Starting with v0.10.1, Moria Agent ships an Apple Silicon (arm64) build only. The last release with an Intel .dmg was v0.10.0.

Download:

  1. Download and run the installer
  2. Follow the installation prompts (installs to %LOCALAPPDATA%\Programs\Moria Agent)
  3. Launch from Start menu or desktop shortcut

Download:

  1. Download the AppImage
  2. Make it executable: chmod +x Moria-Agent-x64.AppImage
  3. Run: ./Moria-Agent-x64.AppImage

Note

FUSE required. Install with:

  • Debian/Ubuntu: sudo apt install fuse libfuse2
  • Fedora: sudo dnf install fuse fuse-libs

Launch

After installation:

  • macOS — Applications folder or Spotlight (CMD+Space, type "Moria Agent")
  • Windows — Start menu, desktop shortcut, or moria-agent in terminal
  • Linux — Run moria-agent from terminal (if installed via script) or run the AppImage directly

First launch opens the setup wizard.
