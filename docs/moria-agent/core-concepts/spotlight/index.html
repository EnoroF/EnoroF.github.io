Spotlight

Open Moria from anywhere, bring in your current application context, and send work to a new or existing session.

Spotlight is the quick-entry surface for Moria Agent. Call it up while you are working in another application, describe what you need, and send the request without leaving your current flow.

Note

Spotlight is currently available on Windows. Moria Agent must be running so the Launcher can connect to your active workspace.

Open Spotlight

Press WIN + SPACE from any application to open Spotlight. If it is unavailable, the Launcher falls back to CTRL + SPACE, then ALT + SPACE.

Spotlight opens centered on the monitor containing your pointer and stays above other windows while active.

Send a request

Type an instruction and press ENTER, or select the arrow button.

By default, Spotlight creates a new temporary Moria session and sends your instruction to it. When Spotlight was opened from File Explorer, the current Explorer directory becomes the new session's working directory.

After submission, Spotlight clears the input and hides so you can continue working. The new or updated session remains available in Moria Agent.

Tip

Send mode requires an instruction. Captured context enriches your request but is not submitted by itself.

Bring in your current context

Each time you open Spotlight with the global shortcut, it checks the active application for useful context. Depending on the application, this can include:

  • The application name
  • The current file, project, page, or location
  • Text selected in the active application
  • Files and folders selected in File Explorer
  • The current Explorer directory

Available context appears above the input. It is appended to your instruction as a clearly separated context section, including selected text or selected paths when available.

Select the close button beside the context to exclude it from the current request. Opening Spotlight again refreshes the context from whichever application is active at that time.

Choose a session

Spotlight can create a new session or continue an existing one.

Press SHIFT + TAB in Send mode to cycle through:

  1. A new session
  2. The last session used by Spotlight
  3. Up to five recent sessions
  4. Back to a new session

When an existing session is selected, a tag such as Last or the session title appears beside the input. Your next request is sent to that session instead of creating another one.

To return to a new session:

  • Press SHIFT + ESC, or
  • Press BACKSPACE while the input is empty and a session tag is active

Search the web

Press TAB to switch between:

  • Send mode — sends an instruction to Moria Agent
  • Search mode — opens a temporary ChatGPT search in your browser

In Search mode, Spotlight searches using your typed query. If selected text was captured from the active application, it is included in the search. You can also search captured selected text without typing an additional query.

Press TAB again to return to Send mode.

Clear or dismiss Spotlight

ESC removes one layer at a time:

  1. Clear the current input
  2. Remove captured application context
  3. Clear the selected session target
  4. Hide Spotlight

Spotlight also hides automatically when it loses focus. Click and drag its background if you want to reposition it before submitting.

ShortcutAction
WIN + SPACEOpen Spotlight on Windows; falls back to CTRL + SPACE, then ALT + SPACE
ENTERSubmit the current instruction or search
TABToggle between Send and Search modes
